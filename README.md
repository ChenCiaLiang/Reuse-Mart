# Reuse-Mart

Reuse-Mart is a simple web-based marketplace where users can browse products, add items to the cart, make purchases, and manage store items.  
The name "Reuse" reflects the idea of reusing or reselling items to reduce waste.

- **PHP/Laravel Backend - Frontend** (`ReuseMart-Web`)
- **Mobile Flutter** (`ReuseMart-Mobile`)
